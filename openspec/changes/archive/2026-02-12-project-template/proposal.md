# Project Template

**Goal**: E2
**ROADMAP**: Phase 3.1

## Problem

Developers need a quick way to scaffold a hybrid Avalonia + WebView app with correct structure and best practices. Manual setup is error-prone and time-consuming.

## Solution

Create `agibuild-hybrid` `dotnet new` project template with Desktop/Bridge/Tests structure. Template includes GreeterService demo, SPA hosting setup, and MockBridgeService test example. Framework parameter (vanilla/react/vue) allows frontend scaffolding choice.
