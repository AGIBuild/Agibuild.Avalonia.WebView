## Roadmap Notes (Phase 5 Alignment)

### In Scope (This Increment)

- Phase `5.2`: 系统集成入站事件纳入 typed capability + policy-first。
- Phase `5.3`: 入站/出站系统集成均提供结构化诊断字段，满足机器可判定。
- Phase `5.4`: app-shell 模板演示双向流（web->host command + host->web event）。
- Phase `5.5`: 增加 CT matrix、governance marker、automation round-trip 证据。

### Out of Scope (Explicit Boundary)

- **不追求 Electron full API parity**（如插件生态、自动更新全链路、安装器生态）。
- 不扩展多宿主（WPF/WinForms/MAUI）兼容目标。
- 不引入 fallback/dual-path 兼容实现。
