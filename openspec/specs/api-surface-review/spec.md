# API Surface Review Spec

## Overview
Pre-1.0 audit of public API consistency, naming, and completeness.

## Requirements

### AR-1: Public type inventory
- All public types documented with members

### AR-2: IWebView consistency
- Identify gaps between IWebView and concrete implementations

### AR-3: Naming audit
- Verify .NET naming conventions

### AR-4: Action items
- Documented with priority and breaking change assessment
